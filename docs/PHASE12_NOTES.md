# PHASE12_NOTES.md

## What this phase adds

Phase 12 adds safe integration patch helpers.

### Included
- integration patch generator
- routes patch output
- services patch output
- sidebar patch output
- bootstrap require patch output

## Why it matters

A generated module now comes with the integration guidance needed to wire it into the framework with less manual effort.

## Remaining manual step

A developer still reviews and pastes the patch output into the real framework files.
